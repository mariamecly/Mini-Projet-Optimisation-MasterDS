# Mini-Projet Optimisation - Master Data Science

Ce dépôt contient la résolution du mini-projet d'optimisation (Janvier 2026).

## Contenu du projet
Le projet est réalisé sous forme de **Notebook Python** et traite les points suivants :
1. **Maximisation du bénéfice** (Programmation linéaire avec Pyomo/GLPK).
2. **Plan de production hebdomadaire** (Mali NEGUE).
3. **Descente de gradient** (Implémentation, tests de pas et visualisations 3D).
4. **Analyse des extrémums** (Calcul des points critiques).

## Outils utilisés
* Python 3
* Pyomo & Solveur GLPK
* Matplotlib & NumPy (pour les graphiques)
